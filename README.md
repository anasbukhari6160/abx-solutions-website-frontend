# ABX Solutions Website

This project is a business website developed for ABX Solutions. It presents company services, information, and contact details in a clean and professional layout.

##  Technologies Used
- HTML5
- CSS3

##  Features
- Modern and clean homepage
- Services section
- About company section
- Contact information
- Responsive layout

##  Project Structure
- index.html
- about.html
- services.html
- contact.html
- style.css
- images/

##  Purpose
This website is developed as a practice and academic project to demonstrate front-end development skills.

##  Deployment
The website can be deployed using Netlify or GitHub Pages.
